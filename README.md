# CRUDS - Book Management System

A simple web app to manage books using JavaScript.

## Features
- Add new book
- Display all books
- Search books by name
- Update book info
- Delete a book
- Store data in localStorage

## Technologies
- HTML
- CSS
- JavaScript (Vanilla)
- localStorage
